# Веб-фронтенд (Vite + React + TypeScript)

Фронтенд — это панель мониторинга для проверки работы API платформы водоснабжения.

## Быстрый запуск

```bash
cd frontend
npm install
npm run dev
```

По умолчанию фронт ожидает доступный backend на `http://localhost:8000/api`. URL можно переопределить через переменную окружения.

Создайте файл `.env.local` в каталоге `frontend` и задайте адрес API:

```
VITE_API_URL=http://localhost:8000/api
```

## Основные возможности

- Авторизация: форма входа использует `/api/users/token` и сохраняет JWT.
- Дашборд: показывает реальные данные пользователей, активов, жалоб, уведомлений, качества воды и спроса.
- Управление активами, жалобами, уведомлениями, прогнозами — каждое представление делает живые запросы к соответствующим REST‑эндпоинтам backend.

Если API возвращает пустые данные, компоненты отображают дружелюбный пустой стейт — это помогает быстро убедиться, что соединение установлено, даже без наполненной базы.
