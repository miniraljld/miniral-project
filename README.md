# Веб-платформа мониторинга и управления системами водоснабжения

## Описание

Веб-платформа для мониторинга, анализа и управления системами водоснабжения и санитарии, разработанная для хакатона «От источника к решению: цифровой путь к воде».

## Архитектура

- **Backend**: Python (FastAPI)
- **Frontend**: React + Vite
- **База данных**: PostgreSQL
- **ORM**: SQLAlchemy
- **API**: RESTful API

## Функциональность

1. **Мониторинг и контроль**

   - Данные инфраструктуры водоснабжения (утечки, давление, температура)
   - Качество воды в реальном времени
   - Состояния санитарных сооружений

2. **Управление и планирование**

   - База данных активов (объектов водоснабжения) с координатами
   - Модуль прогнозирования спроса и планирования распределения воды
   - Инструменты для управления инвестициями

3. **Механизмы взаимодействия с обществом**

   - Форма подачи жалоб и обращений
   - Панель с тарифами и способами оплаты
   - Модуль уведомлений для клиентов

4. **Санитария**
   - Учёт качества воды при любых условиях
   - Система отчётности и мониторинга состояния санитарии
   - Поддержка инклюзивных решений (для уязвимых групп)

## Требования

- Python 3.8+
- Node.js 18+
- PostgreSQL

## Установка и запуск

### Backend

1. Установите зависимости:

   ```bash
   cd backend
   pip install -r requirements.txt
   ```

2. Настройте переменные окружения (создайте файл `.env`):

   ```
   DATABASE_URL=postgresql://Asul_owner:npg_g2bpls0ruyVc@ep-steep-art-a8v9mw8n-pooler.eastus2.azure.neon.tech/Asul?sslmode=require
   SECRET_KEY=your-secret-key-here
   ```

3. Запустите сервер:
   ```bash
   rn main:app --relo uvicoad
   ```

### Frontend

1. Установите зависимости:

   ```bash
   cd frontend
   npm install
   ```

2. Запустите приложение:
   ```bash
   npm run dev
   ```

## API Документация

После запуска сервера документация будет доступна по адресу:

## Переменные окружения

### Backend

- `DATABASE_URL` - URL подключения к базе данных
- `SECRET_KEY` - Секретный ключ для JWT
- `ACCESS_TOKEN_EXPIRE_MINUTES` - Время жизни токена доступа (в минутах)

### Frontend

- `REACT_APP_API_URL` - URL API сервера
